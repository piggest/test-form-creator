# テスト回答欄作成ツール

ブラウザで動作するテスト回答欄作成ツールです。小学校・中学校のテスト用回答欄を簡単に作成できます。

## 特徴

- **サーバー不要**: `index.html` をブラウザで開くだけで動作
- **階層構造**: 大問 → 問 → 回答欄の柔軟な構造
- **縦書きモード**: 縦書き・原稿用紙形式の解答用紙
- **答え表示機能**: 回答欄に正解を入力し、表示/非表示を切り替え可能
- **PDF/印刷対応**: A4サイズでの印刷・PDF出力
- **データ保存**: JSON形式でテストデータを保存・読み込み

## 回答欄タイプ

| タイプ | 説明 |
|--------|------|
| 記号回答式 | 記号で回答（ア、イ、ウなど） |
| 記述式 | 文字数・行数を指定可能 |
| 数値記述式 | 単位付き、比率、時間形式に対応 |
| 原稿用紙形式 | マス目で文字数を指定 |

## 使い方

1. `index.html` をブラウザで開く
2. 「段落を追加」で大問を作成
3. 各段落に回答欄や子段落を追加
4. 「プレビュー」モードで確認
5. 印刷またはPDF保存

## 縦書きモード

「縦書きモード」にチェックを入れると、縦書きレイアウトで解答用紙を作成します。

- 縦書きの原稿用紙形式
- 右から左へのレイアウト
- A4横向き印刷

## 動作環境

- モダンブラウザ（Chrome、Firefox、Safari、Edge）
- JavaScript有効
