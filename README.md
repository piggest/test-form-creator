# テスト回答欄作成ツール

ブラウザで動作するテスト回答欄作成ツールです。選択式・記述式の両方に対応し、国語の縦書き解答用紙も作成できます。

## 特徴

- **サーバー不要**: `index.html` をブラウザで開くだけで動作
- **国語モード**: 縦書き・右から左への解答用紙レイアウト
- **多様な回答形式**: 選択式、記号回答、語句回答、記述式など
- **文字数制限**: 原稿用紙形式での文字数指定が可能
- **PDF/印刷対応**: A4サイズでの印刷・PDF出力
- **データ保存**: JSON形式でテストデータを保存・読み込み

## ファイル構成

```
test-form-creator/
├── index.html      # メインHTML
├── style.css       # スタイルシート
├── js/
│   ├── utils.js    # ユーティリティ関数
│   ├── state.js    # 状態管理
│   ├── render.js   # 描画関連（通常モード）
│   ├── vertical.js # 国語モード描画
│   ├── sections.js # 大問関連
│   ├── questions.js# 問・小問関連
│   ├── subitems.js # 子回答欄関連
│   ├── export.js   # PDF/JSON出力
│   └── main.js     # 初期化・イベントリスナー
├── app.js          # レガシー（単一ファイル版）
└── README.md       # このファイル
```

## 使い方

1. `index.html` をブラウザで開く
2. 「大問を追加」ボタンで大問を作成
3. 各大問に小問（回答欄）を追加
4. 「プレビュー」モードで確認
5. 印刷またはPDF保存

## 回答形式

| 形式 | 説明 |
|------|------|
| 選択式 | 2〜4択の選択問題 |
| ○×式 | 正誤問題 |
| 記号回答式 | 記号で回答（ア、イ、ウなど） |
| 語句回答式 | 短い語句で回答（文字数指定可） |
| 短文記述式 | 1行程度の記述 |
| 長文記述式 | 複数行の記述 |
| 複数回答欄 | 1つの小問に複数の回答欄 |

## 国語モード

「国語モード」にチェックを入れると、縦書き・右から左のレイアウトで解答用紙を作成します。

- 縦書きの原稿用紙形式
- 複数ページ対応
- A4横向き印刷

## 動作環境

- モダンブラウザ（Chrome、Firefox、Safari、Edge）
- JavaScript有効
